### Client-Side Requirements

React hooks will not function on the client side without a proper build step for the components. To enable client-side hook support, you must use a build plugin such as [`frame-master-plugin-apply-react`](https://github.com/shpaw415/frame-master-plugin-apply-react/tree/main) in your project.

**Recommended setup:**

```bash
# Install the required plugin
bun install frame-master-plugin-apply-react
```

This plugin handles the necessary component transformations to ensure hooks work correctly during client-side rendering.
